Como Usar o Preloader
Funções JavaScript Disponíveis:
  • showPreloader(texto) - Mostra preloader principal
  • hidePreloader() - Esconde preloader principal
  • showCardPreloader(elemento) - Preloader para cards
  • hideCardPreloader(elemento) - Remove preloader do card
  • showSkeletonLoader(tabela, linhas) - Skeleton para tabelas
  • showButtonLoader(botao) - Preloader para botões
Recursos Automáticos:
  • Preloader automático no carregamento da página
  • Interceptação automática de requisições AJAX
  • Preloader automático para formulários
  • Preloader automático para navegação
  • Múltiplos estilos e animações
  • Totalmente responsivo
